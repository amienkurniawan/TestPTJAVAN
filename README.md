## Bagaimana cara menjalankan project

pastikan untuk menginstall composer di komputer, ketika perintah "composer install" untuk menginstall package yang ada di project, kemudian import database "family_trees.sql" di aplikasi mysql, file database sudah ada didalam folder project, dan setting .env menyesuaikan dengan DB_USERNAME, DB_PASSWORD, DB_DATABASE 

masuk ke folder project kemudian buka terminal dan jalankan perintah "php artisan serve"
